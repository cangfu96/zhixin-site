# 知心儀官網首頁（靜態網頁）

這是「知心儀」AI輔助循環健康設備的一頁式介紹網站，使用 HTML + CSS 製作，適合部署於 GitHub Pages。

## 📄 網站預覽
👉 https://cangfu96.github.io/zhixin-site/

## 📦 內容說明

- `index.html`：主頁內容
- `style.css`：樣式檔案

## 🚀 如何部署到 GitHub Pages？

1. 登入 GitHub，建立新專案 `zhixin-site`
2. 上傳 `index.html` 和 `style.css`
3. 點選「Settings」→ 「Pages」
4. 選擇 **Deploy from branch**  
   - 分支選擇：`main`  
   - 資料夾選擇：`/ (root)`
5. 按下儲存，等候 30 秒
6. 你會看到網站出現在：  
   `https://cangfu96.github.io/zhixin-site/`
